# 📸 Kechlen Studio

Welcome to **Kechlen Studio**, an interactive Photo Studio experience right in your browser! Built with modern web technologies, Kechlen Studio transforms your standard webcam into a premium photo booth with aesthetic filters, gesture detection, and dynamic effects.

## ✨ Features

- **🌸 Mood Themes**: Customize your experience with beautiful themes like Sakura, Cozy Café, Rainy Night, Neon City, and Minimal White.
- **✋ AI Gesture Control**: Take pictures effortlessly! Show a **Thumbs Up (👍)** to trigger the capture timer, or a **Peace Sign (✌️)** to automatically apply portrait beauty filters.
- **🎀 Photo Frames**: Frame your memories instantly with styles like Polaroid, Kawaii, and Film Strip.
- **📸 Photo Strip Mode**: An AI photoshoot mode that guides you through 4 poses and compiles them into a beautiful, downloadable photo strip.
- **✨ Live Particles**: Interactive screen effects (cherry blossoms, bubbles, stars) that match your chosen mood.
- **📊 AI Scorecard**: Get fun, pseudo-AI ratings for Lighting, Smile, and Composition after taking a photo.

## 🚀 How to Run

1. Clone this repository.
2. Open the project folder.
3. Start a local server (because it requires camera access and fetches assets). For example, using Python:
   ```bash
   python -m http.server 8000
   ```
4. Open your browser and go to `http://localhost:8000`.
5. Allow camera access and enjoy!
